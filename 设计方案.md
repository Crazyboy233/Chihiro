# 千寻记账与日程管理应用设计方案

## 一、项目概述

### 1.1 项目背景
千寻是一款专注于个人财务管理和日程安排的移动应用，首先实现记账功能，并为后续的日程管理和打卡任务预留扩展接口。

### 1.2 技术选型
- **开发框架**: Flutter
- **目标平台**: Android (优先)
- **数据存储**: SQLite (本地数据库)
- **状态管理**: Provider 或 Riverpod
- **架构模式**: MVVM
- **日历集成**: Android Calendar Provider (优先支持小米)

---

## 二、功能模块设计

### 2.1 核心功能模块

#### 2.1.1 记账模块 (当前优先实现)
- **账单记录**
  - 添加收入/支出账单
  - 选择收支类别
  - 添加类别备注（例如类别是购物，可以添加具体备注，短袖，零食等）
  - 输入金额
  - 选择日期
- **账单分类**
  - 预设常用分类：
      - 餐饮、交通、购物、地铁、蔬菜、水果、零食、运动、娱乐、通讯、住房、游戏、长辈、社交、日用、旅行、数码、学习、AI
      - 工资、奖金，理财，兼职，其他
  - 支持自定义添加分类
  - 分类图标管理
  - 分类颜色设置
- **账单查询**
  - 按日期（周，月，年）范围查询
  - 按分类查询
  - 按金额范围查询
  - 搜索功能（按备注搜索）
- **账单统计**
  - 月度收支统计
  - 分类占比饼图
  - 收支趋势折线图
  - 年度统计汇总
- **账单管理**
  - 编辑账单
  - 删除账单
  - 批量操作

#### 2.1.2 预留模块（后续扩展）

- **日程管理模块**
  - 要求可以直接拉取读取手机日历。首先支持小米。
  - 日程添加/编辑/删除
  - 日程提醒
  - 日程分类

- **打卡任务模块**
  - 自定义添加打卡目标
  - 习惯打卡
  - 任务完成记录
  - 打卡统计

### 5.2 核心接口设计

#### 5.2.1 数据服务接口 (DatabaseService)
```dart
abstract class DatabaseService {
  // 账单操作
  Future<int> insertTransaction(Transaction transaction);
  Future<int> updateTransaction(Transaction transaction);
  Future<int> deleteTransaction(int id);
  Future<List<Transaction>> getTransactions({
    DateTime? startDate,
    DateTime? endDate,
    int? categoryId,
    String? searchNote,
  });
  Future<Map<String, double>> getSummaryByDateRange(DateTime startDate, DateTime endDate);
  Future<Map<String, double>> getCategorySummary(DateTime startDate, DateTime endDate, String type);
  
  // 分类操作
  Future<int> insertCategory(Category category);
  Future<int> updateCategory(Category category);
  Future<int> deleteCategory(int id);
  Future<List<Category>> getCategories(String type);
  Future<List<Category>> getAllCategories();
  Future<void> initializeDefaultCategories();
  
  // 预留 - 日程操作
  Future<int> insertSchedule(Schedule schedule);
  Future<int> updateSchedule(Schedule schedule);
  Future<int> deleteSchedule(int id);
  Future<List<Schedule>> getSchedulesByDateRange(DateTime startDate, DateTime endDate);
  
  // 预留 - 打卡操作
  Future<int> insertHabitGoal(HabitGoal goal);
  Future<int> updateHabitGoal(HabitGoal goal);
  Future<int> deleteHabitGoal(int id);
  Future<List<HabitGoal>> getActiveHabitGoals();
  Future<int> insertHabitRecord(HabitRecord record);
  Future<List<HabitRecord>> getHabitRecords(int goalId, DateTime startDate, DateTime endDate);
  
  // 初始化
  Future<void> initialize();
}
```

#### 5.2.2 预留 - 日历服务接口 (CalendarService)
```dart
abstract class CalendarService {
  // 检查权限

  Future<String?> insertEventToCalendar(Schedule schedule);
  Future<bool> updateEventInCalendar(Schedule schedule);

}
```
